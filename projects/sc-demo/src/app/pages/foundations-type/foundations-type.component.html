Tipografía

Escala redonda en rem (root 16), step-named (mismo idioma que el Kit Pro y los devs) y desacoplada de la escala de espaciado. 8 tamaños + 7 line-heights.

    @for (s of sizes; track s.step) {

  • {{ s.sample }}

    font-size-{{ s.step }} {{ s.px }}px · {{ s.rem }}rem
    • }

Line-heights

    @for (l of lineHeights; track l.step) {
  • line-height-{{ l.step }} {{ l.px }}px · {{ l.rem }}rem
    • }

Naming por step (100…800), no por valor: el nombre no miente si el valor cambia, y Figma, código y los devs hablan el mismo idioma. En rem para que la letra escale con el tamaño base del navegador (accesibilidad). Desacoplada de --sc-scale: tipografía y espaciado son ejes independientes. Dos pesos: Regular y Semibold.
