# Icono de carga (búsqueda)

Prototipo de Marta Recio (jul 2026). Icono animado de "lupa buscando" (círculo +
dos arcos girando de forma sincronizada) con el texto "Cargando resultados..." y
un efecto shimmer, recreado en vectorial a partir de una grabación de pantalla.

Copia reunida en el repo (nido temporal de exploraciones, ver la página `/lab` de
sc-demo). Aún NO forma parte del design system.

## Ficheros aquí

- `search_loading_icon.gif` — GIF en alta resolución con el texto horneado como
  píxeles, para pegar directo (Figma reproduce GIFs animados de forma nativa). Es
  lo que muestra la página `/lab`.
- `search_loading_icon.lottie` — dotLottie: solo el icono (círculo + arcos) como
  capas vectoriales editables; el texto se añade como capa nativa en el motor
  final (After Effects, lottie-web, etc.).

> El repositorio original de la designer incluía además el script `build.py` que
> regenera estos formatos y las fuentes Inter. No se copiaron aquí: esto es solo
> el resultado, para preservar la exploración.
